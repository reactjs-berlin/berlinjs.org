ReactJS.berlin is a usergroup focused on ReactJS and related topics. We meet regularly each month at 7p.m. @bitcrowd office, Oranienstr. 6 in Berlin-Kreuzberg. Talks start at 7:30pm.

React Basic Workshop on 4th August 2018

React Basic Workshop

Manjula Dube

If you are beginner to React come join the workshop to start your react with me. We will learn about ES6 basics, Virtual DOM, JSX, Stateless Components, Stateful Components, Props, States, Lifecycle Methods, React, Snapshot testing in React. We will create a simple application along the learning so we have something working at the end of the workshop.

If you want to learn React and have never tried it. This workshop is for you. Come join me. Limited seats.
